# Vibrant Prism UI Protocol v3.1
**Objective:** Maintain a high-status, Noir-Tech aesthetic with zero bloat and maximum responsiveness.

## 1. Technical Infrastructure
*   **Base Styling:** Use **Pico.css** for class-less, semantic HTML foundations. It provides the "Elegant Noir" baseline out of the box.
*   **Design Tokens:** Use **Open Props** for precise control over gradients, shadows, and spacing.
*   **Animation:** Standardize on **GSAP** for high-fidelity entering/exiting transitions.

## 2. Visual Identity (Noir-Tech)
*   **Typography:** 
    *   **Arabic:** **Cairo**. Use `Black (900)` or `Bold (700)` for emphasis.
    *   **English:** **JetBrains Mono**. Maintains the "Terminal/Code" authority.
    *   **Rendering:** Wide letter-spacing on titles; strictly `#D4AF37` (Gold) or `#e2e8f0` (Off-white).
*   **Background:** Deep-space black (`#050505`) with subtle radial gradients.
*   **Accents:** 
    *   Onyx Gold (`#D4AF37`) for authority markers.
    *   Neon Blue (`#00f2ff`) for technical/active signals.
    *   Prism Blur: Layered glassmorphism (`backdrop-filter: blur(20px)`).

## 3. Human-First Interaction
*   **Bilingual Native:** Every UI component must support the Ar/En switcher logic.
*   **Anti-AI Copy:** All labels and buttons must follow the Anti-AI Writing Protocol (Direct, active verbs only).
